## 🚀 Pourquoi importer mes données ?

L'importation vous permet de :

- Conserver votre historique de visionnage accumulé sur plusieurs années.
- Retrouver vos notes et évaluations personnelles.
- Synchroniser vos listes de suivi pour ne rater aucun film.
- Tirer parti des statistiques détaillées de DejaVu dès le premier jour.

## 🛠️ Sources supportées

Nous supportons actuellement l'importation depuis les services suivants :

| Service      | Méthode     | Données récupérées                                |
| :----------- | :---------- | :------------------------------------------------ |
| **IMDb**     | Fichier CSV | Notes, Watchlist, Listes personnalisées           |
| **Trakt.tv** | API / OAuth | Historique complet, Notes, Watchlist, Collections |
| **TMDb**     | API         | Favoris, Watchlist, Notes, Listes                 |
